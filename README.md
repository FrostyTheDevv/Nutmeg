## Available Commands
- `/play` - Play music from YouTube or Spotify
- `/queue` - View/manage queue
- `/volume` - Control volume
- `/loop` - Set loop mode
- `/playback` - Pause, resume, skip, stop
- `/247` - 24/7 mode toggle
- `/help` - Command list
